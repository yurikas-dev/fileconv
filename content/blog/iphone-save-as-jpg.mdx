iPhoneはデフォルトでHEIC形式で写真を保存しますが、設定を変更することでJPG形式で保存できます。WindowsやAndroidとファイルをやりとりする機会が多い方に特におすすめです。

## 設定変更の手順

### カメラのフォーマット設定を変更する

1. iPhoneの **「設定」** を開く
2. **「カメラ」** をタップ
3. **「フォーマット」** をタップ
4. **「互換性優先」** を選択

「互換性優先」に変更するとJPG形式で写真が保存されるようになります。「高効率」がHEICです。

## 設定変更のメリット・デメリット

| | 高効率（HEIC） | 互換性優先（JPG） |
|---|---|---|
| ファイルサイズ | 小さい | 大きい |
| 画質 | 高い | 良い |
| Windows対応 | 要変換 | そのまま開ける |
| Android対応 | 要変換 | そのまま開ける |
| LINEで送れる | 自動変換される | そのまま送れる |

## Windowsへの転送時だけJPGにする方法

カメラのフォーマットはHEICのままにして、Windowsに転送するときだけJPGに変換したい場合は、iPhone側の転送設定を変更します。

1. 「設定」→ **「写真」** をタップ
2. 下にスクロールして **「MACまたはPCに転送」**
3. **「自動」** を選択

「自動」にするとMacへはHEIC、WindowsへはJPGで自動的に転送されます。

## 既存のHEIC写真はどうする？

設定変更後に撮影した写真はJPGになりますが、**過去に撮影したHEICファイルはそのままHEICで保存されています。**

過去の写真をJPGに変換したい場合はFileConvを使うと複数ファイルを一括で変換できます。ファイルはブラウザ内で処理されるため、大切な写真を外部サーバーに送信する必要はありません。

## どちらの設定がおすすめか

| 状況 | おすすめ設定 |
|---|---|
| iPhoneだけで写真を管理している | 高効率（HEIC）のまま |
| Windowsとよくやりとりする | 互換性優先（JPG）に変更 |
| ストレージを節約したい | 高効率（HEIC）のまま |
| とにかく互換性を重視したい | 互換性優先（JPG）に変更 |

## まとめ

- 「設定」→「カメラ」→「フォーマット」→「互換性優先」でJPG保存に変更できる
- Windowsへの転送時だけJPGにしたい場合は「設定」→「写真」→「自動」を選択
- 過去のHEICファイルは変換ツールで一括変換できる
