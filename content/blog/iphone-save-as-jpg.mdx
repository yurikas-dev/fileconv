iPhoneはデフォルトでHEIC形式で写真を保存しますが、設定を変更することでJPG形式で保存できます。WindowsやAndroidとファイルをやりとりする機会が多い方、フリマアプリや仕事でよく写真を使う方に特におすすめの設定です。

## なぜiPhoneはHEICで保存するのか

HEICはJPGと比べてほぼ同じ画質で約半分のファイルサイズを実現できます。iPhone内のストレージを効率的に使うためAppleが標準フォーマットとして採用しましたが、WindowsやAndroidでは標準対応していないため、やりとりの際に不便が生じることがあります。

## 設定変更の手順

### カメラのフォーマット設定を変更する

1. iPhoneの **「設定」** を開く
2. **「カメラ」** をタップ
3. **「フォーマット」** をタップ
4. **「互換性優先」** を選択

「互換性優先」に変更するとJPG形式で写真が保存されるようになります。「高効率」がHEICです。設定後すぐに有効になり、この設定以降に撮影した写真はJPGで保存されます。

## 設定変更のメリット・デメリット

| | 高効率（HEIC） | 互換性優先（JPG） |
|---|---|---|
| ファイルサイズ | 小さい（効率的） | 大きい（約2倍） |
| 画質 | 優秀 | 良い |
| Windows対応 | 要変換 | そのまま開ける |
| Android対応 | 要変換 | そのまま開ける |
| LINEで送れる | 写真送信時は自動変換 | そのまま送れる |
| フリマアプリ | アプリ側で変換される場合あり | 確実に使える |

## ストレージへの影響

JPGに変更するとファイルサイズが約2倍になるため、iPhone内のストレージ消費が増えます。たとえばHEICで1枚2MBだった写真がJPGでは4MB程度になります。月に100枚撮影する場合、差は1年で約2.4GB。iPhoneのストレージ容量が少ない方は注意が必要です。

**ストレージが心配な方向け：** 後述する「転送時のみJPGに変換する」方法がおすすめです。iPhone内はHEICのまま効率的に保存しつつ、Windowsへの転送時だけJPGに変換できます。

## Windowsへの転送時だけJPGにする方法

カメラのフォーマットはHEICのままにして、Windowsに転送するときだけJPGに変換したい場合は、iPhone側の転送設定を変更します。

1. 「設定」→ **「写真」** をタップ
2. 下にスクロールして **「MACまたはPCに転送」**
3. **「自動」** を選択

「自動」にするとMacへはHEIC、WindowsへはJPGで自動的に転送されます。iPhoneのストレージ効率を保ちながら、Windowsとのやりとりもスムーズになる最もバランスのよい設定です。

**注意点：** この設定はUSBケーブルで接続した場合のみ有効です。AirDrop・メール・LINEでの直接共有には適用されません。

## 既存のHEIC写真はどうする？

設定変更後に撮影した写真はJPGになりますが、**過去に撮影したHEICファイルはそのままHEICで保存されています。** 設定を変えても過去の写真は自動的にJPGに変換されません。

過去の写真をJPGに変換したい場合はFileConvを使うと複数ファイルを一括で変換できます。ファイルはブラウザ内で処理されるため、大切な写真を外部サーバーに送信する必要はありません。

## AirDropで送るとHEICになる問題

「互換性優先」に設定していても、iPhoneからMacやiPadにAirDropで送ると相手の端末の設定によってはHEICで受信されることがあります。

AndroidやWindowsにAirDropは使えないため問題になりにくいですが、AirDrop受信側のMacで「設定」→「一般」→「AirDropとHandoff」→「写真の転送はJPG」を有効にしておくと確実にJPGで受け取れます。

## どちらの設定がおすすめか

| 状況 | おすすめ設定 |
|---|---|
| iPhoneだけで写真を管理・ストレージを節約したい | 高効率（HEIC）のまま |
| Windowsと頻繁にやりとりする | 互換性優先（JPG）に変更 |
| フリマアプリをよく使う | 互換性優先（JPG）に変更 |
| バランスを重視したい | HEIC + 転送設定を「自動」に |
| 過去の写真をまとめてJPGにしたい | FileConvで一括変換 |

## まとめ

- 「設定」→「カメラ」→「フォーマット」→「互換性優先」でJPG保存に変更できる
- ストレージを節約しつつWindowsへの転送だけJPGにするなら「設定」→「写真」→「MACまたはPCに転送」→「自動」
- 過去のHEICファイルは変換ツールで一括変換できる
- どちらの方法も組み合わせて使うと最も使い勝手がよい
