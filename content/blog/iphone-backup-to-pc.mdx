iPhoneのストレージが「いっぱい」になる前に、写真をパソコンにバックアップしておくことが大切です。方法はいくつかありますが、HEICファイルの扱いによって手順が変わります。WindowsとMac、それぞれの方法を解説します。

## バックアップ方法の比較

| 方法                    | Windows | Mac | 速度            | HEICの変換           |
| ----------------------- | ------- | --- | --------------- | -------------------- |
| USBケーブル（直接転送） | ○       | ○   | 速い            | 設定による           |
| iCloud写真              | ○       | ○   | Wi-Fi速度に依存 | されない（HEIC保持） |
| Google フォト           | ○       | ○   | Wi-Fi速度に依存 | サーバー側で変換     |
| iTunes（Finder）        | ○       | ○   | 速い            | されない（HEIC保持） |

---

## 方法①：USBケーブルで直接転送（最も手軽）

### Windowsへの転送

1. USBケーブルでiPhoneをパソコンに接続する
2. iPhoneの画面に「このコンピュータを信頼しますか？」が表示されたら「信頼」をタップ
3. Windowsのエクスプローラーを開く
4. 「PC」→「Apple iPhone」→「Internal Storage」→「DCIM」→ 番号付きフォルダを開く
5. 写真ファイルを選択してコピー＆ペースト

**HEIC問題：** このままコピーするとHEIC形式のままWindowsに保存されます。Windowsで開く場合は「HEIF Image Extensions」のインストールか、FileConvでJPGへの変換が必要です。

**JPGで転送したい場合：**
「設定」→「写真」→「MACまたはPCに転送」→「**自動**」を選択すると、Windows接続時に自動でJPGに変換して転送されます。

### Macへの転送

Macの場合は「イメージキャプチャ」アプリを使うのが最も簡単です。

1. USBケーブルでiPhoneを接続
2. 「イメージキャプチャ」を開く（Spotlight検索で「イメージキャプチャ」）
3. 左のサイドバーからiPhoneを選択
4. 読み込みたい写真を選択して「読み込む」をクリック

MacはHEICに対応しているため、HEIC形式のままでも問題なく開けます。

## 方法②：iCloud写真で自動バックアップ

iCloudを使うと、Wi-Fi接続時に自動で写真がバックアップされます。

### 設定手順（iPhone）

1. 「設定」→「自分の名前」→「iCloud」→「写真」
2. 「このiPhoneを同期」をオン

### パソコンからアクセスする方法

**Windows：** [iCloud.com](https://www.icloud.com) にサインインして「写真」を開く、またはWindowsの「iCloud」アプリをインストール

**Mac：** 「写真」アプリが自動的にiCloudと同期される

### 注意点

- **無料ストレージは5GBまで**（写真が多い場合は有料プランが必要）
- 写真はHEIC形式で保存される（iCloud.comからダウンロードするとJPGに変換される場合もある）
- iPhoneのストレージ節約のため「ストレージを最適化」をオンにすると、iPhone本体の写真は低解像度のサムネイルになる

## 方法③：Google フォトでバックアップ

Googleアカウントがあれば無料で使えます（15GBまで）。WindowsでもMacでも手順は同じです。

### 設定手順

1. iPhoneに「Google フォト」アプリをインストール
2. Googleアカウントでサインイン
3. 「バックアップ」をオンにする（Wi-Fi接続時に自動でアップロード）

### パソコンからアクセス

ブラウザで [photos.google.com](https://photos.google.com) を開いてサインインするだけで全写真を閲覧・ダウンロードできます。

### 特徴

- **HEIC→JPGに自動変換**されて保存される（オリジナルHEICは保持されない場合がある）
- ダウンロードするとJPGまたはWebP形式で取得できる
- 無料15GBを超えると有料（Google One）が必要
- iPhoneとAndroidを両方使っている場合に特に便利

## 方法④：Finderを使ってまるごとバックアップ（Mac）

MacのFinderを使うと、写真だけでなくiPhone全体をバックアップできます。

1. USBケーブルでiPhoneを接続
2. Finderを開き、サイドバーからiPhoneを選択
3. 「一般」タブ→「このMacにiPhoneのすべてのデータをバックアップする」を選択
4. 「今すぐバックアップ」をクリック

このバックアップはiPhone全体（写真・連絡先・アプリのデータなど）を含む暗号化されたアーカイブです。写真だけを個別に取り出すには向いていませんが、機種変更前のバックアップとして最も確実です。

---

## HEICで保存されている写真をJPGにまとめて変換したい場合

パソコンにバックアップしたHEICファイルをまとめてJPGに変換したい場合は、FileConvが便利です。

- 複数ファイルを一括でJPGやPNGに変換できる
- ファイルはブラウザ内で処理されるためサーバーに送信されない
- 変換と同時にEXIF情報（GPS位置情報など）の削除も可能

## どの方法を選ぶべきか

| こんな方に                          | おすすめ方法                        |
| ----------------------------------- | ----------------------------------- |
| 今すぐバックアップしたい            | USBケーブルで直接転送               |
| 自動で常に最新状態にしたい          | iCloud写真 または Google フォト     |
| WindowsとAndroidを両方使っている    | Google フォト                       |
| Appleエコシステムに統一している     | iCloud写真                          |
| 機種変更前に確実に保存したい        | Finder（Mac）でまるごとバックアップ |
| HEICで保存した画像をJPGに変換したい | FileConvで変換                      |

## まとめ

- iPhoneの写真バックアップはUSBケーブル・iCloud・Google フォトの3つが主な方法
- **Windowsへの転送時にJPGで受け取りたい場合**は、設定→写真→「自動」に変更する
- HEICのまま保存した場合はFileConvでまとめてJPGに変換できる
- 大切な写真は複数の場所に保存しておくのが安心（ローカル＋クラウドの併用）
