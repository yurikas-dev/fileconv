スマートフォンで撮影した写真をオンラインに投稿するとき、あなたは見えない情報も一緒に公開しているかもしれません。それが **EXIF情報** です。知らないうちに自宅の住所や行動パターンが特定されるリスクがあり、特にフリマアプリやブログを使う方には知っておいてほしい知識です。

## EXIFとは何か

EXIF（Exchangeable Image File Format）とは、写真ファイルに埋め込まれているメタデータのことです。写真の「見た目」とは別に、撮影に関するさまざまな情報がファイルの中に記録されています。

主に記録される情報は以下の通りです。

| カテゴリ | 具体的な情報 |
|---|---|
| 位置情報 | GPS緯度・経度・高度 |
| 日時 | 撮影日時・タイムゾーン |
| カメラ情報 | メーカー・機種名・シリアル番号 |
| 撮影設定 | 露出・ISO・焦点距離・フラッシュの有無 |
| ソフトウェア | 使用したアプリ・OSのバージョン |

これらの情報は、写真を見ただけではわかりません。しかし専用のツールやアプリを使えば誰でも簡単に読み取ることができます。

## なぜプライバシーリスクになるのか

特に問題になるのが **GPS位置情報** です。

iPhoneやAndroidのカメラは、設定によって撮影時の位置情報を写真に記録します。「設定 → プライバシー → 位置情報サービス → カメラ」で「このAppの使用中のみ許可」または「常に許可」になっていると、すべての写真にGPS座標が埋め込まれます。

自宅で撮影した写真をSNSやフリマアプリに投稿した場合、EXIF情報を読める人には撮影場所（＝自宅の住所）が特定できてしまいます。写真の見た目からは場所がわからなくても、EXIFには正確な座標が記録されているためです。

具体的なリスクとしては以下が挙げられます。

- **フリマアプリで商品写真を投稿** → 自宅の場所が特定される
- **旅行中の写真を投稿** → 不在中であることと行き先が同時に判明する
- **子どもの写真を投稿** → 通っている学校や公園の場所が特定される
- **仕事場で撮影した写真を共有** → 職場の場所が特定される

## EXIFはどうやって確認するのか

写真のEXIF情報は、専用ツールや一部のサービスで確認できます。

### Windowsの場合
画像ファイルを右クリック →「プロパティ」→「詳細」タブで確認できます。GPS座標が記録されている場合は「GPSの緯度」「GPSの経度」という項目が表示されます。

### Macの場合
写真アプリで画像を開き「情報を見る」（⌘+I）から確認できます。また、Finder でファイルを右クリック →「情報を見る」でも確認可能です。

### スマートフォンの場合
iPhoneでは「写真」アプリで画像を開き、下にスワイプすると撮影場所が地図で表示されます。これはEXIFのGPS情報をもとに表示しています。

### オンラインツールの場合
「EXIF viewer」などのキーワードで検索すると、画像をアップロードしてEXIF情報を確認できるサービスが見つかります。ただし、プライバシー上の懸念がある写真はアップロードしない方が無難です。

また、GoogleマップのURLと組み合わせると、GPS座標から正確な住所を特定できてしまいます。

## SNSはEXIFを自動で削除してくれる？

TwitterやInstagram、Facebookなど多くのSNSは、アップロード時にEXIF情報を自動的に削除するよう設定されています。そのためSNSへの投稿については比較的安全です。

ただし、SNSへ投稿してもEXIFが残るケースも報告されています。特にLINEでの画像送信では、相手に送った画像にEXIFが残ることがあります。

一方、以下のような場面では**自動削除されないケース**があります。

- フリマアプリ（メルカリ・ラクマなど）
- ブログや個人サイトへの直接アップロード
- メールへの添付
- クラウドストレージの共有リンク（Dropbox・Google Driveなど）
- Discord、Slackなどのビジネス・コミュニティツール

## EXIF情報を削除する方法

### Windowsの標準機能
ファイルを右クリック →「プロパティ」→「詳細」タブ →「プロパティや個人情報を削除する」をクリックします。ただし、すべてのEXIF項目を一括削除できない場合があります。

### Macの標準機能
「プレビュー」アプリで画像を開き、「ツール」→「インスペクタを表示」→「Exif」から一部の情報を確認・削除できます。完全な削除には限界があります。

### 変換ツールを使う
FileConvでは、HEICをJPG/PNGに変換しながらEXIF情報も同時に削除できます。「EXIF情報を削除する」オプションがデフォルトでONになっており、チェックを入れたまま変換するだけでGPS情報を含むすべてのEXIFが除去されます。変換とEXIF削除を一度に行えるため手間がかかりません。

## カメラアプリの位置情報をオフにする

根本的な対策として、カメラアプリの位置情報アクセスをオフにする方法もあります。

**iPhoneの場合：**
「設定」→「プライバシーとセキュリティ」→「位置情報サービス」→「カメラ」→「なし」を選択

**Androidの場合：**
「設定」→「アプリ」→「カメラ」→「権限」→「位置情報」→「許可しない」を選択

ただしこの設定をするとGPS情報が記録されなくなるため、撮影場所の記録が残りません。旅行の思い出写真などで場所を記録しておきたい場合は、アルバムや写真アプリの地図機能を活用する方法と使い分けるのがよいでしょう。

## まとめ

- EXIFとは写真に埋め込まれたメタデータで、GPS情報や撮影日時などが含まれる
- 自宅や職場で撮影した写真には、場所を特定できる座標が記録されている
- SNS以外（フリマアプリ・メール・ブログなど）では自動削除されないケースが多い
- WindowsやMacの標準機能でも削除できるが、変換ツールを使うと一括で対応できる
- カメラアプリの位置情報をオフにするのも有効な対策

写真を共有する前にEXIF情報を確認・削除する習慣をつけることで、意図しない個人情報の流出を防げます。
